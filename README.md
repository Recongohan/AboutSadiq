## Overview

Single-page personal portfolio for Sayed Sadiq — Software Engineer & DevOps Professional based in Dubai, UAE. Built with zero frameworks and zero dependencies. Pure vanilla stack, shipped fast.

---

## Features

- Animated particle network canvas (vanilla JS)
- Typewriter effect with cursor blink
- Scroll-triggered fade-in animations via Intersection Observer API
- Fully responsive across desktop, tablet, and mobile
- Sticky nav with smooth scroll and mobile hamburger menu
- DevTools detection & right-click protection
- Neon blue dark theme with subtle glow effects

---

## Tech Stack

| Layer | Technology |
|---|---|
| Markup | HTML5 |
| Styling | CSS3 (custom properties, grid, flexbox) |
| Logic | Vanilla JavaScript (ES6+) |
| Version Control | Git + GitHub |
| Hosting | Hostinger (static deployment) |

---

## Deployment
```bash
git add .
git commit -m "update portfolio"
git push origin main
```

Source controlled via Git. Changes pushed to GitHub and deployed to Hostinger via static file hosting.

---

## Local Development

No build tools, no package manager, no config files.
```bash
git clone https://github.com/Recongohan/AboutSadiq.git
cd AboutSadiq
# open index.html in your browser
```
